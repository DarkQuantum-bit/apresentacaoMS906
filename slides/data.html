Descrição dos Dados

O conjunto de dados utilizado é proveniente do Comprehensive Earthquake Catalog (ComCat) do USGS, contendo mais de 3 milhões de registros entre 2001 e 2023.

  • Variáveis: data, hora, latitude, longitude, magnitude, profundidade, tsunami.
  • Fontes: USGS e repositório do Kaggle “Ultimate Earthquake Dataset (1990–2023)”.
  • Critério: eventos com magnitude ≥ 2.0, recorte temporal 2001–2023.
